# 國中會考自然科歷屆試題與電子書章節分析 — 待辦清單

## 任務目標

收集最近五年的國中教育會考自然科歷屆試題，下載到專案指定資料夾，並與使用者提供的國中自然電子書章節內容比對，整理出會考出題比例、重點單元、每年常考單元，最後規劃暑假兩個月的七、八年級自然複習進度與每日學習單。

## 資料夾規劃

在專案資料夾內建立：

```text
G:\我的雲端硬碟\my-classroom-tools\國中\
G:\我的雲端硬碟\my-classroom-tools\國中\國中會考資料夾\
G:\我的雲端硬碟\my-classroom-tools\國中\國中會考資料夾\自然科目資料夾\
```

參考電子書位置：

```text
G:\我的雲端硬碟\my-classroom-tools\國中\電子書\自然\
```

## 待辦清單

### 1. 資料夾與來源檢查

- [ ] 確認 `國中` 資料夾是否已存在。
- [ ] 確認 `國中\電子書\自然` 是否已有使用者提供的電子書檔案。
- [ ] 建立 `國中\國中會考資料夾\自然科目資料夾`。
- [ ] 列出電子書檔案名稱、年級、冊次、章節可讀性。

### 2. 上網搜尋並下載最近五年會考自然科試題

- [ ] 搜尋最近五年國中教育會考自然科試題與答案。
- [ ] 優先使用官方或可信來源。
- [ ] 下載試題、答案、解析，如有 PDF 或 Word 檔都保存。
- [ ] 依年份命名檔案，例如：
  - `2026_國中會考_自然_試題.pdf`
  - `2026_國中會考_自然_答案.pdf`
- [ ] 存放到：

```text
國中\國中會考資料夾\自然科目資料夾\
```

### 3. 試題整理

- [ ] 逐年整理題號、題型、主題、可能對應章節。
- [ ] 建立題目索引表：
  - 年份
  - 題號
  - 題型
  - 知識點
  - 對應年級
  - 對應電子書章節
  - 難度
  - 是否跨章節
- [ ] 如 PDF 可解析，盡量抽取文字；若掃描品質差，標記需人工處理。

### 4. 與電子書章節比對

- [ ] 讀取 `國中\電子書\自然` 中的電子書。
- [ ] 建立七、八、九年級自然章節目錄。
- [ ] 比對歷屆試題與電子書章節。
- [ ] 統計每章節出題次數與比例。
- [ ] 標記：
  - 高頻單元
  - 每年必考或幾乎必考單元
  - 跨冊整合單元
  - 學生常錯概念

### 5. 分析報告

- [ ] 產出一份分析報告，內容包含：
  - 最近五年自然科出題分布
  - 七、八、九年級各冊比例
  - 高頻單元排行
  - 每年常見題型
  - 需要優先補救的概念
  - 對低成就學生的學習建議
- [ ] 建議輸出為 Markdown，必要時再轉成 Word 或 PDF。

### 6. 暑假兩個月複習規劃

- [ ] 以今年暑假兩個月為期程，規劃七、八年級自然複習。
- [ ] 以每日進度安排：
  - 當日複習單元
  - 觀念建立
  - 會考例題
  - 類題練習
  - 解答與說明
- [ ] 每週設計一次總複習或小測。
- [ ] 課程設計要適合低成就、低動機學生：
  - 低門檻開始
  - 短題組
  - 明確步驟
  - 圖像化或生活化說明
  - 可逐步撤除鷹架

### 7. 每日學習單

- [ ] 每日建立一份學習單。
- [ ] 每份學習單包含：
  - 今日觀念說明
  - 一題會考歷屆試題
  - 一題仿會考類題
  - 解答
  - 詳細說明
  - 學生自我檢核
- [ ] 題目需標註來源年份與題號。
- [ ] 類題不得直接複製原題，需保留同概念、改變情境或數據。

### 8. 輸出成果

- [ ] 建立總分析報告。
- [ ] 建立暑假兩個月每日複習進度表。
- [ ] 建立每週每日學習單。
- [ ] 將所有成果整理在：

```text
國中\國中會考資料夾\自然科目資料夾\
```
